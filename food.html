1. Applied Design Tools and Interfaces

This project focuses on combining the knowledge of graphic-design software with the theories of web usability and interface creation.
This project required me to research and design screens for a web and mobile app. I chose a food app that was geared toward single parents/guardians with little time to fix healthy meals.
After the initial design, I did a user test and then went back and made adjustments to the screens according to the user feedback
